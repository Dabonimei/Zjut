## 主要考核方式
期末考核的成绩将参考 GitHub 上大家的活跃程度和做出的贡献
同时也会参考成员平时在精弘社团中的活跃度
总之，是为了鼓励大家多多学习积累知识，多和其他人交流，这同时也会对你的能力有很大的提升。
